# Projeto de Visão Computacional - Fase 6 (FarmTech Solutions)

Este repositório contém o desenvolvimento de um sistema de detecção de objetos utilizando a arquitetura YOLOv5 para a disciplina de IA.

## 📁 Estrutura do Projeto
- `dataset/`: Imagens de treino, validação e teste (organizadas conforme padrão YOLO).
- `data.yaml`: Configuração das classes e caminhos.
- `notebooks/`: Arquivo .ipynb com todo o processo de treinamento e análise.

## 📈 Resultados e Conclusões
Foram realizadas duas simulações:
1. **30 Épocas:** Equilíbrio entre tempo de processamento e precisão inicial.
2. **60 Épocas:** Melhora significativa na curva de aprendizado e detecção de objetos menores.
